# Neo4j 可视化与截图指引（作业2）

## 1. 启动 Neo4j

如果你用 Docker：

```bash
docker run --rm \
  --name neo4j-assignment2 \
  -p 7474:7474 -p 7687:7687 \
  -e NEO4J_AUTH=neo4j/password \
  neo4j:5
```

然后浏览器打开：`http://localhost:7474`

## 2. 写入图谱数据

先确保已生成 `outputs/nodes.csv` 与 `outputs/edges.csv`：

```bash
conda run -n ml python src/prepare_dataset.py
conda run -n ml python src/extract_kg.py
```

写入 Neo4j：

```bash
conda run -n ml python src/load_neo4j.py --uri bolt://localhost:7687 --user neo4j --password password
```

## 3. 在 Neo4j Browser 运行查询并截图

建议截图至少 2 张（贴到 `报告.md` 的“可视化截图”处）：

### 截图 1：整体图谱（限定 50 条边）

```cypher
MATCH (d:Disease)-[r]->(e) RETURN d,r,e LIMIT 50;
```

### 截图 2：某个疾病中心子图

把 `高血压` 换成你想展示的疾病名：

```cypher
MATCH (d:Disease {name:"高血压"})-[r]->(e)
RETURN d,r,e;
```

### （可选）截图 3：症状最多的疾病 Top10

```cypher
MATCH (d:Disease)-[:HAS_SYMPTOM]->(s:Symptom)
RETURN d.name AS disease, count(*) AS n
ORDER BY n DESC LIMIT 10;
```
