## 2. 图像边缘检测（Sobel）

### 2.1 方法说明

Sobel 算子通过对图像在 x/y 方向的梯度做近似求导来刻画边缘强度。实验流程为：

1. 彩色图转灰度；
2. 高斯平滑（抑制噪声）；
3. 分别计算 x/y 方向 Sobel 梯度并求梯度幅值；
4. 对幅值图归一化，并使用 Otsu 自动阈值二值化得到边缘图。

### 2.3 现象分析

- 对规则图形（2.2.1）边缘检测效果清晰，主要边界（矩形、圆、直线、文字轮廓）都有明显响应。
- 对真实拍摄图（2.2.2）文字、投影边界、人物轮廓会产生大量梯度响应，边缘图更“密”，说明 Sobel 对纹理与细节同样敏感。
- 模糊（2.2.3）会削弱高频信息，使得边缘更平滑但更粗，细小结构更容易消失；整体边缘数量减少。
- 加噪（2.2.4）会引入随机梯度响应，边缘图中出现较多散点噪声；因此实际应用中通常需要更强的去噪、非极大值抑制或形态学处理来提升可读性。

## 3. 图像分类：机器学习方法 vs 深度学习方法（MNIST）

本实验选择 MNIST 手写数字数据集（10 类，28×28 灰度图），分别实现：

- 机器学习方法：PCA + Linear SVM
- 深度学习方法：CNN（卷积神经网络）

并对比两者的分类准确率与计算效率（训练/推理或评估耗时）。

### 3.1 机器学习方法：PCA + Linear SVM

#### 3.1.1 方法与参数

- 特征：将 28×28 灰度图展平为 784 维向量
- 预处理：标准化（StandardScaler）
- 降维：PCA，维度 = 64
- 分类器：LinearSVC（线性 SVM）
- 训练/测试样本：训练 20000，测试 10000

#### 3.1.2 实验结果

指标（来自本次运行结果）：

- 准确率：0.8958
- 训练耗时：6.43 s
- 推理耗时：0.04 s（单张约 0.00393 ms）

### 3.2 深度学习方法：CNN

#### 3.2.1 网络结构与参数

本实验使用简单 CNN（输入 1×28×28）：

- Conv(1→16, 3×3) + ReLU + MaxPool(2×2)
- Conv(16→32, 3×3) + ReLU + MaxPool(2×2)
- Flatten
- FC(32×7×7→128) + ReLU
- FC(128→10)

训练设置：

- 优化器：Adam，学习率 1e-3
- Epoch：2
- Batch size：128
- 训练/测试样本：训练 60000，测试 10000
- 设备：CPU

#### 3.2.2 实验结果

指标（来自本次运行结果）：

- 准确率：0.9868
- 训练耗时：63.43 s
- 评估耗时：1.17 s（单张约 0.11653 ms）

### 3.4 对比分析与讨论

1) **准确率**  
CNN（0.9832）显著高于 PCA+Linear SVM（0.8958）。原因在于：SVM 主要基于展平后的像素做线性判别（PCA 仅做线性子空间压缩），对局部结构、平移不变性等视觉先验刻画不足；CNN 通过卷积学习局部特征并逐层组合，能更好表达笔画形状与空间结构，因此在 MNIST 上更容易获得更高精度。

2) **训练与推理效率**  
在 CPU 环境下，SVM 训练与推理明显更快：它的训练属于凸优化且参数量较少；CNN 需要多层卷积与反向传播迭代优化，计算量更大，所以耗时更长。若使用 GPU，CNN 的吞吐通常会显著提升，训练/推理时间会明显下降。

3) **可扩展性**  
当数据更复杂（如 CIFAR-10 彩色自然图像）时，展平像素 + 线性模型往往难以取得高精度，通常需要更强的特征工程或更复杂的核方法；CNN 等深度模型能够端到端学习更丰富的视觉特征，通常更适合复杂图像分类任务，但代价是更高的算力需求与调参成本。
